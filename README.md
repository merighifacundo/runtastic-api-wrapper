# Runtastic api Wrapper

#### This RestAPI layer authenticates over the services on runtastic and parse the html to get the authenticationToken and the cookies. Then acts like a rest api.


## Ruby version: '2.4.2'

## System dependencies: 

* MongoDB
* Runtastic

## Database creation

* MongoDB

## Services

* ApplicationJob

## Deployment Instructions
